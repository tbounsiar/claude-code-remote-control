Privacy Policy

Last updated: February 25, 2026

This Privacy Policy describes how Claude Code Remote Control ("the App", "we", "us") handles information when you use our mobile application.

1. Information We Collect

Claude Code Remote Control does not collect, store, or transmit any personal data to our servers. The App functions as a remote control interface between your mobile device and your local Claude Code CLI session.

The only data exchanged is:

  • Commands and responses: Text commands you send and responses received from your local Claude Code session. This data is transmitted directly between your mobile device and your own machine via a local or secure WebSocket connection and is never stored by us.
  • Session tokens: Temporary tokens used to authenticate your remote session. These are generated locally and never stored on external servers.

2. Data Storage

All data remains on your devices. We do not operate any backend servers that store your personal information, commands, or Claude Code session data.

3. Third-Party Services

The App connects to Anthropic's Claude Code service running on your own machine. Your use of Claude Code is governed by Anthropic's Terms of Service and Privacy Policy.

4. Permissions

The App may request the following permissions:

  • Internet access: Required to establish a WebSocket connection to your Claude Code session.
  • Local Network access: Required to connect to Claude Code running on your local machine.

5. Children's Privacy

The App is not directed to children under 13. We do not knowingly collect personal information from children.

6. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be reflected by updating the "Last updated" date above.
